Familiarize Yourself with Browsing Data

Previous | Next | See Also

PRIDE Inspector's main data view allows you to browse and inspect the selected experiment's content.


The main data view is tab-based. When you open a file or PRIDE experiment, a set of tabs will be created. Each tab represents a different aspect of the data source. There are five different tabs:

  • General Tab -this area shows the metadata information about the data source, such as references, sample details, protocol etc.
  • Protein Tab - this displays all protein identifications from the data source. Furthermore, it shows a protein's post-translational modifications as well as the corresponding peptides.
  • Peptide Tab - this area shows all peptides linked to protein identifications. Additionally, the peptide's spectra are being displayed.
  • Spectrum/Chromatogram Tab - this area shows all spectra as well as the chromatogram from the data source.
  • Summary Chart Tab - a list of summary charts to get a quick feeling for the general quality of the investigated experiment.
