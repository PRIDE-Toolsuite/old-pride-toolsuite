Download ProteomeXchange submissions

Previous| Next | See Also

If you are a ProteomeXchange submitter or journal reviewer, you can now download private ProteomeXchange submissions using the user name and password has been given to you.

To start download, you can:
  • Click on the Private Download button in the Welcom Screen
  • Click on on the Main Toolbar
  • Go to menu File | Private Download
Once selected, the private download window will popup. Click on the "ProteomeXchange" tab, you will see it consists of two sections:
  • User login
  • Download submission

User login

If you are a ProteomeXchange Submitter and want to download your own submission, you can use the username and password you have chosen when you register with PRIDE.

If you are a journal reviewer, you should be given a PRIDE reviewer account. Type in the reviewer account username and password.

Click on Login button once finished, PRIDE Inspector will start to retrieve all of your experiments details.

Download submission

You can choose the directory where you want to save your experiments by clicking on the Browse button.

The details of the experiments you can download are shown in the experiment details. The table below gives a short description for every column in the experiment details table:

Column Description
Accession ProteomeXchange accession
ID The ID of the file within the context of this submission
Size(M) The size of experiment file to download in megabyte.
Name File name
Type ProteomeXchange submission type, such as: RESULT, RAW
PRIDE PRIDE experiment accession
Relation Related files
Download Checkbox which you can select which file to download
You can choose whether to open the files after downloading by checking Open after download
