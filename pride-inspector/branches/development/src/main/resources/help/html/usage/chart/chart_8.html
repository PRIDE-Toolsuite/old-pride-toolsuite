Number of Missed Tryptic Cleavages Chart

Previous | Next | See Also

Displays the number of missed tryptic cleavages.

Explanation

A histogram representing the percentage of peptides in the experiment with a different number of missed tryptic cleavages peptides. This graph is only applicable to experiments where trypsin is used.

Two assumptions were made for these calculations: first, the enzyme used in the experiment is trypsin; second, the cleavage rule used by the enzyme is “C-terminal side of K or R except if P is C-term to K or R”.

This chart can highlight problems with the digestion process of the investigated samples. An unexpected high fraction of missed cleavages generally points to an inefficient digestion. The average number of peptides containing missed tryptic cleavages in PRIDE experiments is approximately 25%.

This is an example of a highly efficient digestion with only about 10% of the peptides containing one missed cleavage.
