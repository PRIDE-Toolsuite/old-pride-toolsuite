Understanding Summary Charts

Previous | Next | See Also

In this section, you can find out about what the Summary Charts represents.

  • Peak Intensity Distribution Charge - A histogram of ion intensity vs frequency for all MS2 spectra in a single PRIDE experiment.
  • Precursor Ion Charge Distribution - Displays a bar chart of precursor ion charge for a single PRIDE experiment.
  • Average MS/MS spectrum - Displays the average MS/MS spectrum.
  • Distribution of Precursor Ion Masses - Displays a frequency distribution of product ion m/z for different precursor ion charges.
  • Number of Peptides Identified per Protein - Displays a bar chart with the number of peptides identified per protein for a single PRIDE experiment.
  • Number of Peaks per Spectrum - Displays a histogram of number of peaks per MS/MS spectrum in a single PRIDE experiment.
  • Delta m/z - Displays a relative frequency distribution of identified peptide mass - precursor ion mass.
  • Number of Missed Tryptic Cleavages - Displays the number of missed tryptic cleavages.
