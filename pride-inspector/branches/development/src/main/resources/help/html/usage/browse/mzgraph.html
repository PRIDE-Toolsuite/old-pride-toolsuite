Spectrum/Chromatogram Tab

Previous | Next | See Also

The Spectrum/Chromatogram Tab provides a spectrum/chromatogram centric view. All spectra and chromatogram from the data source are displayed in this area. It consists of two sections:

  • Spectrum/Chromatogram Browser
  • Spectrum/Chromatogram Details

Spectrum/Chromatogram Browser

The Spectrum/Chromatogram Browser displays the spectrum / chromatogram selected in the spectra / chromatogram table below.
For more details, please refer to: Spectrum Browser, Chromatogram Browser.

Spectrum/Chromatogram Details

To save memory space and provide a faster user interface, only the first 200 spectra / chromatograms are initially loaded.

The number of loaded spectra / chromatograms are displayed on the top left of the spectrum/chromatogram details table.

You can click Load Next to load the next 200, or you can load all by clicking Load All

The table below gives a short description for every column in the spectrum table:
Column Description
Row Number Displays the row number
Spectrum ID The original spectrum ID from the data source.
MS Level The MS level of a specific spectrum
Precursor Charge The spectrum's precursor's charge
Precursor m/z The spectrum's precursor's m/z
Precursor Intensity The spectrum's precursor's intensity
Sum of Intensity The sum of all peak intensities in the spectrum
#peaks The total number of peaks in the spectrum
The table below gives a short description for every column in the chromatogram table:
Column Description
Row Number Displays the row number
Chromatogram ID The original chromatogram ID from the data source.
