Trying mzML or PRIDE XML Example Files

Previous | Next | See Also

Two examples files are included in PRIDE Inspector installation by default, you can use them to try out PRIDE Inspector if you don't have any data at your hand.

  • PRIDE XML - this is an small PRIDE public experiment, accession number is 9768. For more examples, you can download them from PRIDE Main Website
  • mzML - this is a small example file from PSI website. For more examples, you can download them from PSI Website
There are two different ways of accessing the example datasets:
  • Click on the two buttons on Welcome Screen
  • Go to the main menu by clicking Examples
