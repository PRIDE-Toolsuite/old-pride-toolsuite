Review Project

Previous| Next | See Also

If you are a PRIDE submitter or journal reviewer, you can now download and review private PRIDE experiments using the user name and password given to you.

To start download, you can:
  • Click on the Review Project button in the Welcom Screen
  • Click on on the Main Toolbar
  • Go to menu File | Review Project
Once selected, the login window will popup. You need to follow the following steps:
  • User Login
  • My Projects
  • Project Download

User Login

If you are a PRIDE Submitter and want to download your own experiments, you can use the username and password you have chosen when you register with PRIDE.

If you are a journal reviewer, you should be given a PRIDE reviewer account. Type in the reviewer account username and password.

Click on Login button once finished, PRIDE Inspector will start to retrieve all of your project details.

My projects

Once login, you should see all your experimental projects listed in "My projects" dialog.

Click on View files button to download project file.

Project Download

The details of the experiments you can download are shown in the experiment details. The table below gives a short description for every column in the experiment details table:

Column Description
File Name The name of the file
Size(M) The size of experiment file to download in megabyte.
Type Experimental file type
Download Checkbox which you can select which experiment to download
You can choose whether to open the files after downloading by checking Open after download
