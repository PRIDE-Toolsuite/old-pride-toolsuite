Decoy Filter

Previous | Next | See Also

Decoy Filter can be accessed from the protein tab, the peptide tab and the quantification tab. Once clicked, a menu will be shown with options (see Figure below).



The decoy filter menu contains four options:

  • Setting: this option will open a decoy filter dialog for setting the filter.
  • Non Decoy: this option will show all the non-decoy proteins.
  • Decoy: this option will show all the decoy proteins.
  • Undo Filter: this option will remove all existing fitler.



The decoy filter dialog contains two required settings:

  • Filter Action: this indicates the location of the decoy filter string.
  • Filter String: the string used for filtering the decoy proteins.

Setting a new decoy filter will trigger a update in experiment summary component, both decoy protein ratio and decoy peptide ratio will be displayed as the result.
