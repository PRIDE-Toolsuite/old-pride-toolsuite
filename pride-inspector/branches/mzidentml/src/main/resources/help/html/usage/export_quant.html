Export Quantitative Data

Previous | Next | See Also

Export Quantitative Data dialog can filter according to quantitative ratios, all the filter results will be shown in the protein table, and you can choose to export these results.
There are three key features in this dialog:

  • Filter - use to set the filter parameters to restrict the results
  • Export - export filtered results into an tab-delimited file
  • Esemble View - show all the filtered proteins in the Ensembl karyotype viewer.
  • Reactome Pathway - show all the filtered proteins with their quantitative ratios in the Reactome pathways.


Filter

Three parameters can be used for the filtering process:

  • Reagent Ratio: choose which type of the reagent ratio to filter, this is essentially an column in the protein table.
  • Regulation: choose whether it is up/down regulated, in the PRIDE Inspector, this means whether the value is above one or below one.
  • Percentage: choose the change percentage, for example, if set to 20% and up regulated, this means any value above or equal to 1.20.

Export

Clicking on this button will open an save to file dialog, this enables you to export the filtered quantitative results into tab-delimited file.

Ensembl Viewer

This will map all the proteins in the protein table to the Ensembl Karyotype Viewer, here is an example.

Reactome Pathway

This will map all the proteins in the protein table to the Reactome pathways, please note that the time to calculate the results depends on the number of proteins and their quantitative values. The same feature can be found on Reactome's website.
