Menus and Toolbars

Previous | Next | See Also

PRIDE Inspector's menus and toolbars provide access to the main functions. The main menu and toolbar contains actions that affect the data content at the file or database level. Some of the actions are duplicated in both the menus and toolbars for quick access.


The main menus contains four sub menus:

  • File Menu
  • Examples Menu
  • Export Menu
  • Help Menu

The main tool bar contains four buttons:

  • Open File Toolbar
  • Connect to PRIDE Public Database Toolbar
  • Private Download Toolbar
  • Help Toolbar

File Menu

File Menu contains commands for opening and closing data sources.

  • Open File - opens single or multiple mzML/MzIdentML/PRIDE XML files.
  • Connect PRIDE Database - connects to the PRIDE public database, displays a list of available public experiments
  • Private Download - downloads private experiments from PRIDE.
  • Close - closes the file or PRIDE experiment you are currently viewing.
  • Close All - closes all currently open files and PRIDE experiments.
  • Exist - exists PRIDE Inspector

Examples Menu

Examples lists two examples to quickly try out PRIDE Inspector, one mzML and on PRIDE XML file. Details about these examples can be found at Trying mzML or PRIDE XML Example Files

Export Menu

Export data from the data source in the current view into various file formats, you can export four different types of data:

  • PRIDE XML - export an experiment into PRIDE XML format file, this option is only available if you are viewing an experiment from PRIDE public instance.
  • Spectra (.mgf) - exports all spectra into a Mascot Generic Format file
  • Spectrum Details - exports spectrum related information, such as: ms level, precursor charge, precursor m/z value and etc, into a tab delimited text file. This file can then be processed using programs like Microsoft Excel.
  • Protein-Peptide Mappings - exports all proteins together with their corresponding peptides into a tab delimited text file. No filtering of the data is done and duplicate peptide entries are exported as well.
  • Protein Details - exports all the protein related information, such as: protein accession, PRIDE protein b1Score, PRIDE protein threshold, number of peptides etc., into a tab delimited text file.
  • Peptide Details - exports all the peptide related information, such as: peptide sequence, precursor charge, delta m/z, etc. into a tab delimited text file.

Help Menu

Help Menu contains links to the help document. It provides additional information about PRIDE Inspector and contains a page to provide us with feedback.

The main toolbar contains four key commands:

  • Open File Toolbar - opens a single or multiple mzML/MzIdentML/PRIDE XML files.
  • Connect PRIDE Public Database Toolbar - connects to the PRIDE database, displays a list of public available experiments
  • Private Download Toolbar - downloads private experiments from PRIDE.
  • Help - gives quick access to help documents.
