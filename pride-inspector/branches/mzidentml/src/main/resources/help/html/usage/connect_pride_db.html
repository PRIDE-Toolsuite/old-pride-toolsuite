Connecting to PRIDE Public database

Previous | Next | See Also

This chapter explains how to connect pride inspector to the public pride database and access the different experiments stored in it.

The database is located at the EBI, Cambridge, and can be publicly accessed through the following address as well: 193.62.194.210 using as username inspector, password inspector and the port 5000.


You can connect the pride inspector to the public database in 3 different ways:

  • From the Welcome screen, clicking on the Connect to PRIDE Database icon
  • From the File | Connect PRIDE database option in the menu bar
  • From the toolbar, clicking on the button

Once you are connected to the database, you can view the different experiments by using the navigation bar in the left hand side:
  • if you know the experiment accession, you can enter it in the search box, and press Enter or click the button
  • you can navigate through the list of all the public experiments by scrolling down the experiments list and selecting the experiment you would like to inspect
