Connecting to PRIDE Public database

Previous | Next | See Also

This chapter explains how to connect pride inspector to the public pride database and access the different experiments stored in it.

The database is located at the EBI, Cambridge, and can be publicly accessed through the following address as well: 193.62.194.210 using as username inspector, password inspector and the port 5000.


You can connect the pride inspector to the public database in 3 different ways:

  • From the Welcome screen, clicking on the Connect to PRIDE Database icon
  • From the File | Connect PRIDE database option in the menu bar
  • From the toolbar, clicking on the button

Once you are connected to the database, a search panel consists of two parts will be displayed:
  • Search Box: In this box, you can specify which metadata field you want to search for. And the criteria for searching, as well as the search string. Once the search button is clicked, the table below will show all the matching experiments.
  • Experiment Table: This table contains all the matching experiments from the search, if no search has been made, all the experiments are listed by default. At the top right corner above the table, the number of visible experiments are shown. You can open the experiment by clicking on the plus sign at the "View" column.
  • Close: You can close the database search panel by clicking on the "Close" button.
