General Tab

Previous | Next | See Also

The General Tab provides a unified view on the metadata of different types of data sources. It is arranged in a series of three panels, each panel showing a specific type of metadata. The type of metadata is indicated by the panel's title. Each metadata panel can be viewed by clicking on the button at the top of the General Tab.


You can find out more about each metadata panels in the sections below:

  • Experiment General
  • Sample & Protocol
  • Instrument & Processing

Experiemnt General

The overall information about the experiment.
The top section of this panel shows a summary of the key metadata, this include experiment title, project name, sample species, sample tissues, mass spec instrument and a short experiment description.
References of this experiment will be displayed if present, along with contact information and additional support information.

Sample & Protocol

Additional information on the sample investigated in the experiment. Both mzML and PRIDE XML format contain this information. It generally contains the species, tissue and cell type investigated in the experiment.

The protocol table gives a detailed description on the experimental steps. It starts with the protocol name, followed by a list of steps. Note that, only PRIDE XML contains this information.

Instrument & Processing

Instrument Configuration captures the mass spectrometer model. It contains the following key sections:

  • Instrument Name - Descriptive name of the instrument (make, model, significant customisations)
  • Source - Invariant ion source (e.g. MALDI) information, as a run of name-value pairs
  • Analyzer - Mass analyzer component (e.g. quadrupole, collision cell), ordered so as to reflect the physical order of the described components in the mass spectrometer.
  • Detector - Ion detector information, as a run of name-value pairs.

Data processing shows a description of the default processing by which the peak list(s) were generated, it also displayed the software been used.
