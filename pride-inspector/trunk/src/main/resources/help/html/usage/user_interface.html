Familiarize Yourself with User Interface

Previous | Next | See Also

This chapter explains how PRIDE Inspector's user interface is organised. Reading through the following sections will help you find your way through PRIDE Inspector


When you first run PRIDE Inspector and have no open files or database connections, PRIDE Inspector displays the Welcome Screen, which provides quick access to the main functions.

  • Welcome Screen - provides quick access to the main functions.
  • Menus and Tool Bar - main menu and the toolbars provide access to PRIDE Inspector's functions.
  • Navigation Panel - helps you navigate through the opened files and manages the open database connections
  • Status Bar - displays the status of running background operations
