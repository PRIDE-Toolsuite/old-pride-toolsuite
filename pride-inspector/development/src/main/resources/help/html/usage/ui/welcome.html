Welcome Screen

Previous | Next | See Also

PRIDE Inspector displays the Welcome Screen when no files or PRIDE experiments are opened. From this screen, you can quickly access the major functions of PRIDE Inspector.


The Welcome Screen is divided into four sections:

  • Quick Start
  • Try Examples
  • Learn More About
  • Feedback

Quick Start

Use the buttons in this section to Open mzML/PRIDE XML Files, Connect to the PRIDE Public Database, download private PRIDE experiments and read the help documents.

Try Examples

Use the two buttons in this section to open an mzML or PRIDE XML example file. These files included in the PRIDE Inspector installation.

Learn More About

This section provides quick links to some of the most important topics in the help section.
The topics include: PRIDE, PRIDE XML, PRIDE Public Database, mzML, Private Download, Summary Charts

Feedback

This button opens your local email client, which allows you to write an email to the PRIDE HelpDesk.
