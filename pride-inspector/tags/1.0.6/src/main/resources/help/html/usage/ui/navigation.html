PRIDE Inspector can open multiple files and public experiments from the PRIDE database simultaneously. The Navigation Panel is organising the opened data sources. This panel can be hidden/shown by clicking on the left toolbar.


The Navigation Panel consists of two main sub panels:

  • Data Sources
  • Experiments

Data Sources

This panel is being displayed by default, all opened data sources are shown here. You can view the content of the data source by clicking on the entry in the panel. Furthermore, you can close the data source by clicking on . A separate is created for every opened file or public PRIDE experiment.

Experiments

This panel is hidden by default, it is only shown when a connection to the PRIDE database has been established. Once shown, a list of public experiments is displayed, each experiment identified by its PRIDE accession. To view an experiment, one can click on the accession in the panel, or directly type the accession in the Jump To text box.
